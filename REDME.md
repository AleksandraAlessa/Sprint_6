# Проект Sprint_6: Автотесты для сервиса «Яндекс.Самокат»

## Описание
Проект содержит набор автоматических тестов для учебного сервиса [«Яндекс.Самокат»](https://qa-scooter.praktikum-services.ru/).

Тесты покрывают следующий функционал:
- Раздел «Вопросы о важном» (аккордеон) – проверка текста ответов на все 8 вопросов.
- Позитивный сценарий заказа самоката (с параметризацией: два набора данных, две точки входа – верхняя и нижняя кнопки «Заказать»).
- Переход на главную страницу при клике на логотип «Самоката».
- Переход на главную страницу Дзена (через редирект) при клике на логотип Яндекса.

## Технологии
- **Python** (3.14)
- **Selenium WebDriver** – управление браузером
- **pytest** – запуск тестов
- **Allure** – генерация отчётов
- **webdriver-manager** – автоматическое управление драйверами браузера
- **Firefox** – браузер для запуска тестов (можно заменить на Chrome)

## Установка и запуск
- Создать и активировать виртуальное окружение
**python -m venv .venv
**source .venv/Scripts/activate   # для Windows Git Bash
**или .venv\Scripts\activate    # для Windows CMD
- Установить зависимости
** pip install -r requirements.txt
-  Запустить тесты
** pytest tests/ -v
- Сгенерировать Allure-отчёт
** pytest tests/ --alluredir=allure_results
** allure serve allure_results
